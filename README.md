# Mobile Shop

SPA para compra de dispositivos móviles. Construida con Preact + Vite.

## Requisitos

- Node.js >= 18
- npm >= 9

## Instalación

```bash
npm install
```

## Scripts

```bash
npm start              # Servidor de desarrollo (localhost:5173)
npm run build          # Compilar para producción
npm test               # Ejecutar tests
npm run lint           # Verificar código
```
